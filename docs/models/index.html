    ESM-C auto-discovery

    esmc_300m and esmc_600m use EvolutionaryScale's esm SDK (requires Python ≥ 3.10, conflicts with fair-esm). No manual environment switching needed — the tool scans your conda envs and runs extraction via subprocess when the SDK is found. One-time setup: conda create -n esmc python=3.10 -y && conda activate esmc && pip install esm httpx. After that, just pick esmc_300m or esmc_600m — it works automatically.

    Simple encodings + tabular¶

    • onehot, blosum62 — positional encodings (require aligned, equal-length sequences). The only inputs cnn1d accepts.
    • tabular — your extra numeric/categorical columns, combined with any other source.
